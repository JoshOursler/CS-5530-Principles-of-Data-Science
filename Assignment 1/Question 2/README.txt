The raw data collected is meant to determine the effect of different conditions on a student's performance on basic subject tests.
	In addition to the scores of the math, reading, and writing sections of a test, data on a student's gender, race/ethnicity, 
	parental education level, lunch and whether or not the student completed a test preparation course was collected.

The goal of this study is to specifically determine the effect of gender on test scores, the effect of the test preparation
	course on students' scores, the effect of the lunch chosen on students' scores, the correlation between scores
	in the separate subjects, and the association between math scores and reading scores.

Overall, the purpose of this question was to become familiar with Data Visualization 
	and presenting our findings with these plots and charts.

The code first opens four libraries used to complete the data visualization process.
	The pandas library which was used to ingest the raw data into a dataframe that can be used by python.
	The numpy library for any numerical analysis that needs to be performed.
	The matplotlib.pyplot library for making plots look presentable and save to the computer well.
	The seaborn library for more plots to use for more clear analysis.

The code then create the 5 visuals that are present in the reports folder. Each visual has the dimension 800x600 px and 300 DPI.
