The data in the raw data folder represents data collected from 10 female participants to test 
	the relation between reduced grip strength and a higher frailty score in females.

Height is measured in inches, weight in pounds, age in years, grip strength in kilograms, 
	and frailty as either a yes (Y) or a no (N).

Overall, the purpose of this question was to become more familiar with the process of handling data.
	Specifically, data ingestion, processing, and analysis. In addition, the project was meant to
	be organized like a data science experiment.

The code is divided up into 3 sections using Jupyter Notebooks.
	Data ingestion, which was taking a csv file in the project folder and reading the data into
		a dataframe from the pandas python library.
	Data processing, which was taking the raw data and transforming it into data that is standarized
		and more readily used for analysis.
	Data analysis, which was using the cleaned data to determine if our hypothesis 
		that grip strength correlates to frailty was true or not.
